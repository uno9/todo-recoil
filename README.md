 # 💻 Title
todo-recoil

## ✅ Requirements
- React
- Recoil
- Typescript

Install :
```bash
$ npx create-react-app todo-recoil --typescript
```

## 🚀 Usage
```bash
$ git clone https://github.com/uno9/todo-recoil.git
$ cd HANSHIN-Tigers-report

$ npm install recoil
$ npm run start
```
